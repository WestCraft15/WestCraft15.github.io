Troubleshooting Guide

If a Key isn't spawning...

First, make sure you have all the previous Keys. Then, talk to GPB in the Void Room and try again. If it still doesn't spawn, you've already collected it.

If a Key is inaccessible/uncollectable...

It's probably fake. Click here for a list of fake Keys.

If you can't find an area...

See below or click here.

If GPB won't say the right thing...

Just keep trying, you’ll get it eventually. Also, be sure to leave the room and come back after talking to him 6 times in a row.

If you can't access Champion's Road...

Simply P rank The Crumbling Tower of Pizza.
